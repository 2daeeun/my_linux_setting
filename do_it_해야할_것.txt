네트워크 모니터링       : sudo iftop -i wlp3s0
무선랜 연결             : nmtui
시스템 모니터링         : htop  -> 설정 파일 스크립트 짜기
                          conky -> 설정 파일 스크립트 짜기

zsh                     : powerlevel10k 스크립트 짜기
배터리 체크 명령어      : acpi
밝기 조절 명령어        : brightnessctl
                            brightnessctl set 100%
                            brightnessctl set 50%
사운드 조절             : alsamixer
                            amixer set 'Master' 10%+
                            amixer set 'Master' 10%-
                            amixer set 'Master' mute
                            amixer set 'Master' unmute
